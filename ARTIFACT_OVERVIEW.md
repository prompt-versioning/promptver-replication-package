# Artifact Overview

The artifact is organized around the evaluation pipeline:

1. Prompt edit catalog
2. Executor outputs
3. Paraphrastic noise-floor calibration
4. Stable witness filtering
5. Hard-violation checks
6. Empirical label assignment
7. PROMPTVER and baseline scoring
8. Reproduction of paper tables

The package does not require re-running commercial LLM APIs to reproduce the paper tables.  
All raw model outputs used in the paper are included.  
Scripts that call external APIs are provided for completeness but are not needed for the main reproduction.
